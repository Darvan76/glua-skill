# ai_schedule

---

Allows you to set up a schedule for NPCs to follow. This lets you dictate their actions.

---


## Miembros (1)


---

### `ai_schedule:New` `[server]`

Creates a schedule for scripted NPC.

**Argumentos:**

- `string name` — Name of the schedule.

**Retorna:**

- `table` — A table containing schedule information to be used with ENTITY:StartSchedule.
