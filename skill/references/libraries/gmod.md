### `gmod:GetGamemode` `[client/server]`

Returns the GAMEMODE table.

You will want to wait until GM:OnGamemodeLoaded for it to return a valid value.

**Argumentos:**

*(sin argumentos)*

**Retorna:**

- `table{GM}` — The `GAMEMODE` table.
