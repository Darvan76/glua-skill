### `video:Record` `[client/menu]`

Attempts to create an IVideoWriter.

**Argumentos:**

- `table config` — The video config. See Structures/VideoData.

**Retorna:**

- `IVideoWriter` — The video object. (returns **false** if there is an error)
- `string` — The error string, if there is an error.
